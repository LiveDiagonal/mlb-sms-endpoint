# MLB SMS Endpoint

## Running Locally

Make sure you have [Node.js](http://nodejs.org/) installed.

```sh
$ git clone git@github.com:livediagonal/mlb-whim-app.git # or clone your own fork
$ cd mlb-whim-app
$ npm install
$ npm start
```

Your app should now be running on [localhost:5000](http://localhost:5000/).
